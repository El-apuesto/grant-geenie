## 🚀 How to Use

### For Free Users
1. Complete 9-question eligibility survey
2. Get up to 20 matched grants per month
3. View grant titles, agencies, deadlines, and amounts
4. Upgrade to Pro for unlimited access

### For Pro Users
1. **Smart Filters mode** (default):
   - See grants matched to your profile
   - Automatically filtered by eligibility, fields, budget
   - No expired grants
   - Toggle "Show all grants" to disable filters

2. **Search All mode** (Pro toggle):
   - Browse all 79,000+ active grants
   - Use search bar to find specific keywords
   - Sort by deadline or amount
   - Save favorites

3. **Theme preferences**:
   - Click Sun/Moon icon in navbar (Landing page)
   - Or go to Settings > Appearance
   - Choice persists across sessions

## 🛠️ Technical Details

### Grant Filtering Logic

```typescript
// 1. Remove expired grants
filtered = filtered.filter(grant => !isGrantExpired(grant.close_date, grant.is_rolling));

// 2. Under $250k (unless "Show all" enabled)
filtered = filtered.filter(grant => {
  const ceiling = parseFloat(grant.award_ceiling?.replace(/[^0-9.-]+/g, '') || '0');
  return ceiling <= 250000;
});

// 3. Match funding amount range from questionnaire
// 4. Match primary fields (Arts, Environment, etc.) via keyword search
// 5. Filter out 501(c)(3)-only grants if user has no fiscal sponsor
```

### Database

```sql
-- fiscal_sponsors table
CREATE TABLE fiscal_sponsors (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  name text NOT NULL,
  city text,
  state text,
  focus_areas text[],
  website text,
  fee_structure text,
  eligibility text,
  created_at timestamptz DEFAULT now()
```

## 🚀 Deployment

### Prerequisites
```bash
# Install dependencies
pnpm install  # or npm install

# Set up Supabase
# Run migration: supabase/migrations/20260108_create_fiscal_sponsors_401.sql
```

### Environment Variables
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
VITE_GA_MEASUREMENT_ID=your_ga_id
```

### Dev Server
```bash
pnpm dev
# Open http://localhost:5173
```

### Build for Production
```bash
pnpm build
```
